# Auto-shutdown-AFK
Una aplicacion hecha para apagar la pc forzosamente luego de determinado tiempo. determinado en segundos, pero con ejemplos desde 1 minuto hasta 4 horas. Es mi primer proyecto de programacion, hecho con ayuda de IA. Ideal si tienes que actualizar un juego, renderizar un video, o dejar la pc trabajando mucho tiempo y no quieres que quede encendida.
