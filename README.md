# Auto-shutdown-AFK
Una aplicacion hecha para apagar la pc forzosamente luego de determinado tiempo. contado en segundos, pero con ejemplos desde 1 minuto hasta 4 horas. Es mi primer proyecto de programacion, hecho con ayuda de IA. Ideal si tienes que actualizar un juego, renderizar un video, o dejar la pc trabajando mucho tiempo y no quieres que quede encendida sin hacer nada y consumiendo energia/recursos. personalmente me gusta cuidar la salud de mis componentes, asi que decidi hacer este programa para mi uso personal. pero lo publico por si a alguien mas le sirve, estoy abierto a criticas y puedo agregar funciones.
